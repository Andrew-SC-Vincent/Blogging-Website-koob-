-- Installation Instructions

1. Extract the Project folder to your preferred directory on your local machine.

2. Make sure you have XAMPP Version 8.2.12 or later installed. If not, download and install XAMPP from the official website.

3. Start the Apache and MySQL services from the XAMPP control panel.

4. Open your web browser and navigate to `http://localhost/phpmyadmin`.

5. Import the koob_database.sql file to create database.

----IMPORTANT----

6. Use this URL in the web browser to start application: https://localhost/koob/Project/views/login_page/login.php'
